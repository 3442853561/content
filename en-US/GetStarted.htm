Get Started with Windows IoT

Follow the instructions below to set up or select a board, set up your development environment and get started writing your first app.

Setup a Raspberry Pi 2 or MinnowBoard MAX

If you have a Raspberry Pi 2 or a MinnowBoard MAX, you can setup and configure it easily using the Device Center. Device Center can currently be used to set up on the RTM (public) version of Windows 10 IoT Core and requires a PC running Windows 10.

Please make sure you are running the public release of Windows 10 (version.10.0.10240) or better. If you aren't, you can upgrade from here.
