Welcome to the Adafruit Starter Pack for Windows 10 IoT Core on Raspberry Pi2.

Windows 10 IoT Core is a new edition of Windows targeted towards small, embedded devices and maker boards such as Raspberry Pi2 with or without screens. With Windows 10 IoT Core, you can write a Universal Windows app that is the interface and “personality” for your device. IoT core is designed to work level bus interfaces such as I2C, SPI, USB and enables you to use a variety of programming languages in Visual Studio to rapidly create maker projects on Raspberry Pi2.

Oh, and you can use it to build a light and color sensor, an internet-connected weather sensor and more. The Adafruit Starter Pack with Windows 10 IoT Core includes all you need to get started.

Check out your parts, set up your PC and Programming Environment, set up your Raspberry Pi 2 Board and you'll be ready to make your first project.
