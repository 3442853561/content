Lesson 1: Introduction and Setup

{% include kit-steps.html device=page.deviceName %}

Windows 10 IoT Core is a new edition of Windows targeted towards small embedded devices and maker boards such as Raspberry Pi2. IoT core is designed to work with low-level bus interfaces such as I2C, SPI, USB. You can write a universal Windows App using a variety of programming languages in Visual Studio to rapidly create maker projects.

Oh, and you can use it to read sensor data, control actuators, connect to the cloud, create IoT applications and much more. The Starter Pack from Adafruit includes all the necessary parts to get started.

Check out your parts, set up your PC and Programming Environment, set up your Raspberry Pi 2 Board and you'll be ready to make your first project.

Let's get started!
